About This Project

This is a capstone project for a full stack development course, showcasing the integration of various technologies to build a comprehensive web application.

Project Overview

The application demonstrates modern web development practices using Django as the backend framework, with HTML, CSS, and Bootstrap for the frontend. It includes user authentication, a product store, and notes functionality.

Technologies Used

  • Backend: Django (Python)
  • Frontend: HTML, CSS, Bootstrap
  • Database: SQLite (development), PostgreSQL (production)
  • Authentication: Django's built-in auth system

Features Implemented

Store Management

Full CRUD operations for products with image uploads and categorization.

Notes System

Personal note-taking functionality with organization and search capabilities.
